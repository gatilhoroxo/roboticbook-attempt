## Regra 0
A pergunta vem antes do conceito, o conceito antes da estrutura.
Alimente com o que tiver — mesmo que seja só uma pergunta sem resposta ainda.
Os mapas existem para servir a curiosidade, não o contrário.
### Foco no bottom-up
Isso não é construído de cima pra baixo. Nasce da pergunta, não da estrutura.
Se não existe uma dúvida real por trás, não crie a nota ainda.
A raiz é a curiosidade. Os mapas vêm depois, quando a raiz pede organização.
### Foco em continuidade sem culpa
Uma nota pequena e incompleta > nenhuma nota.
Este cofre não exige perfeição, exige continuidade.
Se travou, é sinal de pré-requisito faltando — não de fracasso.
